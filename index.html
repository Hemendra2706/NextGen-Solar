Intelligent Decision Logic

Our system autonomously monitors environmental factors and executes cleaning cycles based on real-time efficiency drops.

1

Sensing Environment

ESP32 reads Lux levels via BH1750 and measures actual panel current output with precision sensors.

2

Polynomial Analysis

The model calculates expected current using the trained formula:

I = (2.93e-18 * L²) + (5.60e-06 * L) + 4.19e-08
3

Dynamic Trigger

If Measured < Threshold, the cleaning sequence activates automatically (Motor Forward → Brush → Reverse).
